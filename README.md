# Online Voting System (C++)

## Project Description
This is a simple Online Voting System built using C++ that allows multiple users to vote securely with one-person-one-vote rule.

## Features
- Multiple users voting system
- One person one vote
- Admin view for results
- File handling for data storage
- Continuous voting system

## Language Used
- C++

## How it works
Users enter their name, vote for a party, and system stores votes permanently using files.
